About This Dashboard

This dashboard visualizes an artist's Spotify audio features, track trends, top tracks, yearly release patterns, and popularity evolution over time.

Features include:

  • Audio feature analysis (energy, danceability, valence, etc.)
  • Trend analysis across different metrics
  • Top tracks with popularity scores
  • Release timeline and popularity evolution
  • Interactive charts and responsive design
